Sobre a WebBooks

Para este trabalho, desenvolvi um site de hospedagem de livros, que reúne obras de diversas categorias, como romance, terror, ação, entre outras. O principal objetivo do site é proporcionar aos usuários uma navegação intuitiva por gêneros literários, permitindo que, ao clicarem em um dos livros disponíveis da página, sejam redirecionados para uma nova página com o conteúdo completo do livro. No momento, essa funcionalidade encontra-se em fase de estruturação.

A página inicial está organizada por seções, utilizando classes CSS que aplicam uma estilização uniforme a todos os livros exibidos. As obras são agrupadas de acordo com seus gêneros, o que facilita a navegação e a identificação dos conteúdos de interesse.

Além disso, há um botão que leva ao formulário de recomendação, onde o usuário pode selecionar seus gêneros literários favoritos. Essa funcionalidade já está implementada com base nos gêneros escolhidos e poderá, futuramente, ser expandida para realizar recomendações mais personalizadas, considerando também critérios como o tamanho da obra, o status de conclusão (completo ou em andamento), entre outros.

O formulário foi desenvolvido com validação em JavaScript, exigindo que pelo menos uma opção seja selecionada em cada pergunta, e permitindo múltiplas escolhas por questão. O checkbox padrão foi substituído por uma versão personalizada com CSS, tornando o design visualmente atrativo.

Os dados do formulário são enviados por meio do método GET para outra página, onde são validados e armazenados com JavaScript. A partir das categorias selecionadas pelo usuário, o sistema é capaz de recomendar livros, que são exibidos em uma seção específica chamada “Recomendados”.

Todas as páginas do site contam com um menu fixo, que inclui:

  • Link para retornar à página principal (através do nome e logotipo do site);
  • Acesso à página de login (com interface visual já implementada, mas ainda sem funcionalidade de envio, que seria feita via método POST futuramente);
  • Página “Sobre”, com informações sobre o projeto e sua proposta.

Este site foi desenvolvido por Luise Marques Duarte como parte de um projeto acadêmico, com o objetivo de aplicar conceitos de HTML, CSS e JavaScript em um ambiente prático e funcional.
